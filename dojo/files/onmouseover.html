onmouseover & onmouseout

HOME
ABOUT

When you position your mouse over the HOME or ABOUT menu items, the myMouseOverFunction is triggered to set the input text background color to blue and the text color to white for the current menu item.

When you leave the HOME or ABOUT html elements, the myMouseOutFunction is triggered to set the input text background color back to white and the text color to black for the current menu item.

Right-click => Inspect to view the code
